NOA Asia Special

The "NOA Asia Special" project represents my work experience in front-end development and showcases my skills and abilities. Key highlights of the project include:

  • Development of a responsive website using the Angular framework.
  • Utilization of ready-made solutions such as modules, classes, directives, pipes, and guards to build the website's functionality.
  • Implementation of 3 levels of user access: public pages, user-specific pages with data storage for user profiles and orders, and an admin panel for managing products, categories, promotions, and viewing all orders.
  • Integration of Firebase Firestore Database for data storage and management of products, users, and orders.
  • Deployment of the website on Firebase Hosting for online accessibility.
  • Using the Git version control system and hosting the project code on GitHub.

The "NOA Asia Special" project demonstrates my ability to develop functional and visually appealing interfaces using modern technologies. I am ready to contribute to your team and expand my skills through real-world projects.
